# Coding OnePager

## Basic coding interview data structures and algos in plain C

### Implemented features:
* Sort:
  * Insertion Sort
  * Quick sort
  * Heap sort
  * Pancake sort
* Search:
  * Binary search
* Queue
* Graph (adjacency list graph)
* Helpers:
  * power (no math.h required)
  * print array
  * swap elements
  * flip array
  * find max element
